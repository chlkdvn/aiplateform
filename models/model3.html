Introduction

NeuralCore SmartAI Assistant provides a comprehensive suite of programmatic interfaces for natural language processing, automated reasoning, and data synthesis. This guide will walk you through integrating the assistant into your applications.

Onboarding Steps

1

Generate your API Secret

Navigate to the Developer Settings in your dashboard to generate a production-ready key.

2

Install the NeuralCore SDK

We support official libraries for JavaScript, Python, Go, and Ruby.

Authentication

All requests to the NeuralCore API must be authenticated using Bearer tokens in the Authorization header.

import { SmartAI } from '@neuralcore/sdk';

const assistant = new SmartAI({
  apiKey: 'nc_live_51P8pI2K9...',
  environment: 'production'
});

const response = await assistant.generate({
  prompt: 'Analyze current market trends for retail AI',
  context: 'Enterprise Finance',
  stream: true
});
